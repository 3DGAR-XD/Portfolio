¿Quién Soy?

Soy un Desarrollador Front-End Junior, con experiencia en desarrollo de aplicaciones web, usando tecnologías como HTML, CSS y JavaScript, además de tener una gran capacidad de trabajo en equipo. Me gusta pasar el tiempo mejorando mis habilidades y aprendiendo cosas nuevas como el envío de datos mediante archivos JSON, CSV o XML, el manejo de APIs y lógica de programación en videojuegos.

Entre las tecnologías que sé utilizar están: HTML5, CSS3 y Javascript. Además, sé usar programas de edición multimedia como Adobe Photoshop para edición de fotos, Camtasia Studio para edición de video y Audacity para edición de audio. Se hacer mantenimiento preventivo y correctivo a computadores, backups, instalación o desinstalación de programas, limpieza completa de discos duros e instalacion de sistemas operativos. Uso sistemas Linux basados en Debian como Kali Linux, Ubuntu entre otros.

Me gustaría aprender a utilizar frameworks como Vue, Angular o React para aumentar mi productividad, Git y Github para gestión de proyectos, Node.js para adentrarme en el Back-End, y otros IDES y editores de código como Visual Studio, Sublime Text o WebStorm.

¿Qué Hago?

Yo hago la parte visual e interactiva de una aplicación web y su estructura, usando HTML, CSS y JavaScript. Uso diseños web prediseñados, ya que no se crearlos, pero si se manejar el SEO de una web para optimizar su posición en los buscadores.
