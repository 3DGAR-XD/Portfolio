¿Quien Soy?

Soy Edgar de Jesús Luna Chima, desarrollador web junior, técnico en sistemas de información, vivo en Colombia, me gusta la programación de páginas web y videojuegos, soy una persona que le gusta aprender cosas nuevas y hacer las cosas bien.

Empezé a programar desde los 16 años, aprendiendo lo básico de Python, ví que esto me podría ser de ayuda, así que hice scripts para automatizar tareas básicas como eliminar archivos temporales o un ejecutador de aplicaciones, luego intenté hacer videojuegos básicos en 2D usando motores como GameMaker o RPGMaker.

Despues de usar GameMaker logré hacer un videojuego básico en un tiempo récord (1 semana), ya que era para un proyecto escolar, Comenzé con el desarrollo web hace 1 año, cuando recibía clases sobre HTML y CSS básico, entonces decidí hacer una página para aplicar lo aprendido, y de ahí surgió mi alto interés sobre el desarrollo web.

He estado aprendiendo sobre javascript para hacer mis páginas dinámicas y he buscado buenas prácticas de desarrollo web para optimizarlas. Sé que me falta muchas cosas por aprender en el desarrollo web, aunque considero que esta es la mejor que he construido.
